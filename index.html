TahuJS: Framework Pengembangan Aplikasi AI Komprehensif

Mempermudah pembangunan aplikasi berbasis kecerdasan buatan dengan integrasi LLM, alat bawaan, dan layanan kustom.

Pendahuluan

TahuJS adalah framework JavaScript yang dirancang untuk menyederhanakan dan mempercepat pengembangan aplikasi berbasis kecerdasan buatan. Dengan mengintegrasikan berbagai pustaka AI terkemuka dan menyediakan serangkaian alat bawaan yang kuat, TahuJS memungkinkan pengembang untuk fokus pada logika inti aplikasi mereka, bukan pada kompleksitas integrasi AI.

Baik Anda ingin membangun agen cerdas yang dapat berinteraksi secara dinamis, mengotomatiskan pencarian informasi di web, menganalisis data lokasi, atau melakukan perhitungan kompleks, TahuJS menyediakan fondasi yang kokoh dan mudah digunakan.

Fitur Utama

Orkestrasi AI dengan LangChain.js

Manfaatkan kekuatan LangChain.js untuk membangun agen AI yang kompleks, merantai panggilan Large Language Model (LLM), dan mengintegrasikan berbagai alat secara mulus. TahuJS menyediakan abstraksi yang memudahkan Anda dalam mendefinisikan perilaku agen dan alur kerja AI.

Dukungan Multi-Penyedia LLM

Fleksibilitas untuk memilih penyedia LLM favorit Anda. TahuJS mendukung integrasi dengan:

  • **OpenRouter:** Akses berbagai model canggih seperti Claude, GPT, dan Gemini dari satu API.
  • **Google Gemini:** Manfaatkan kemampuan model Gemini langsung dari Google.

Alat Bawaan yang Komprehensif

TahuJS dilengkapi dengan serangkaian alat yang siap pakai untuk berbagai kebutuhan:

  • **Pencarian Web (`webSearch`):** Menggunakan SerpApi, DuckDuckGo, dan scraping Google.
  • **Pencarian Lokasi (`findLocation`):** Menemukan lokasi dengan detail koordinat, elevasi, dan tautan peta.
  • **Petunjuk Arah (`getDirections`):** Memberikan rute perjalanan antara dua titik.
  • **Perhitungan Matematis (`calculate`):** Mengevaluasi ekspresi matematika kompleks.
  • **Web Scraping (`webScrape`):** Mengekstrak konten relevan dari halaman web.
  • **Informasi Tanggal & Waktu (`dateTime`):** Mendapatkan waktu saat ini di berbagai zona waktu.
  • **Ketinggian (`getElevation`):** Mendapatkan data ketinggian untuk koordinat geografis.

Layanan Kustom yang Kuat

Dibangun di atas layanan khusus seperti `SearchService` dan `MapService`, TahuJS menyediakan fungsionalitas pencarian dan pemetaan yang robust dan dapat diperluas, memastikan hasil yang akurat dan relevan.

Validasi Konfigurasi

Dengan `ConfigValidator` bawaan, TahuJS membantu Anda memastikan bahwa semua pengaturan API dan konfigurasi penting lainnya telah diatur dengan benar, meminimalkan kesalahan dan mempercepat proses pengembangan.

Mudah Digunakan & Diperluas

Dirancang dengan modularitas, TahuJS mudah diintegrasikan ke dalam proyek Node.js yang ada dan memungkinkan Anda untuk mendaftarkan alat kustom Anda sendiri, memperluas kemampuannya sesuai kebutuhan spesifik aplikasi Anda.

Ikhtisar Tumpukan Teknologi

TahuJS dibangun di atas fondasi teknologi modern dan teruji:

  • **Core Language & Runtime:** JavaScript (ESM) berjalan di Node.js.
  • **AI Orchestration:** LangChain.js (`@langchain/openai`, `@langchain/google-genai`, `langchain`).
  • **HTTP Requests:** Axios untuk semua panggilan API eksternal.
  • **Web Scraping:** Cheerio untuk parsing HTML.
  • **Mathematical Operations:** Math.js untuk perhitungan.
  • **Terminal Utilities:** `chalk` untuk output berwarna dan `qrcode-terminal` untuk kode QR.
  • **Search Services:** `SearchService` kustom (SerpApi, DuckDuckGo, Google scraping).
  • **Mapping & Location Services:** `MapService` kustom (OpenStreetMap Nominatim, StaticMap, Open-Elevation API, Mapbox).
  • **Configuration Management:** `ConfigValidator` kustom.

Memulai

Untuk memulai dengan TahuJS, ikuti langkah-langkah sederhana ini:

  1. **Pastikan Node.js Terinstal:** TahuJS membutuhkan Node.js versi 18 atau lebih tinggi.
  2. **Kloning Repositori:** 
    git clone https://github.com/yourusername/tahujs.git
cd tahujs
  3. **Instal Dependensi:** 
    npm install
  4. **Konfigurasi API Key:** Edit file konfigurasi Anda (misalnya, di `src/tahu.js` atau di tempat Anda membuat instansi TahuJS) dengan API key yang diperlukan dari penyedia LLM dan layanan lainnya.
  5. **Jalankan Contoh:** Jelajahi folder `example/` untuk melihat bagaimana TahuJS dapat digunakan dalam skenario nyata.
